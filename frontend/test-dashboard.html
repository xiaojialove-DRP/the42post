4. API 端点说明

后端需要实现以下 3 个 API 端点：

1️⃣ GET /api/skills/{soulHash}/impact?token={token}

请求示例：
GET /api/skills/42p_xyz789/impact?token=abc123

响应示例：
{
  "id": "42p_xyz789",
  "title": "Grandma Filter",
  "views": 142,
  "downloads": { "markdown": 23, "langchain": 15, "mcp": 8 },
  "starlight_count": 34,
  "feedback_count": 7,
  "recent_feedback": [
    { "author": "User1", "text": "很实用！", "rating": 5 },
    ...
  ]
}

2️⃣ POST /api/email/send-forge-success

请求示例：
POST /api/email/send-forge-success
Content-Type: application/json

{
  "recipientEmail": "creator@example.com",
  "skillData": { ... },
  "emailHtml": "..."
}

响应：
{ "success": true, "messageId": "..." }

3️⃣ GET /api/skills/{soulHash}/download?format=markdown|langchain|mcp

请求示例：
GET /api/skills/42p_xyz789/download?format=markdown

响应：
返回对应格式的文件（.md / .py / .json）

5. 完整使用流程

  1. 用户在 Skill Forge 工坊创建 Skill
  2. 系统调用 onSkillForgeSuccess(skillData)
  3. 数据保存到 localStorage 和数据库
  4. 邮件发送给 Creator（包含纪念卡片 + 3 个下载按钮）
  5. 首页刷新时，Dashboard 卡片自动显示
  6. Creator 可以：
    • 在首页网格中查看实时数据
    • 点击"View Details"查看完整 Dashboard
    • 在邮件中下载 3 种格式
    • 点击"Share"分享 Skill
