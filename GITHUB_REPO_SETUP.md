## 🔧 GitHub 设置步骤

### 第1步：创建仓库（如果还没有创建）

1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 输入仓库名称：`skill-forging-methodology`
4. 选择 "Public"
5. 点击 "Create repository"

### 第2步：配置仓库信息

1. 进入仓库设置 (Settings)
2. 在 "About" 部分添加：
   - **Description**: `A universal framework for transforming natural language ideas into clear, verifiable, and cross-culturally applicable skills.`
   - **Website**: （如果有的话，暂时可留空）
   - **Topics**: `skill-forging`, `methodology`, `value-alignment`, `ai-ethics`, `design-methodology`, `semantic-capital`
3. 勾选 "Include in the home page" (可选)

### 第3步：推送代码

