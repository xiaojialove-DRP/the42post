## 🎯 Core Concepts (Quick Summary)

### The Five-Layer Skill Structure

Every skill is composed of:

1. **DEFINING** — Core principle (1-2 sentences)
2. **INSTANTIATING** — Real-world examples (before/after)
3. **FENCING** — Boundary conditions (when to apply/not apply)
4. **VALIDATING** — Test cases (verification criteria)
5. **CONTEXTUALIZING** — Cultural perspectives (global adaptations)

### Skill Forge Workflow

```
IDEA (user input)
  ↓ (AI generates structure)
GENERATING (AI creates five-layer framework)
  ↓ (user refines)
FORGE & EDIT (user edits, AI assists)
  ↓ (user confirms)
PUBLISHING (skill goes public, email sent)
```

### Key Features

- **Soul-Hash ID**: Unique, permanent identifier for each skill
- **Creator Card**: Shareable proof of authorship (PNG/PDF)
- **Playground**: Interactive canvas for exploring skills
- **Archive**: Public library of all published skills
